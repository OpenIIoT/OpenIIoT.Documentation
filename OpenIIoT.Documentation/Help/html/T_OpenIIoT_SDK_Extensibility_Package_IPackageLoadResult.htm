IPackageLoadResult Interface
The PackageLoadResult extends the Result class and adds a list of type Package and a Dictionary with key and value types of string. The ValidPackages Package list contains the list of valid plugin archives discovered during the load. The InvalidPackage dictionary contains the list of archives that did not pass validation, along with the reason validation failed.
