Remarks

An [!:IConnector] instance implementing IWriteable is responsible for writing data to [!:ConnectorItem] s by way of the Write(Item, Object) method. This method accepts a ConnectorItem instance and a value boxed in an Object.

The Write() method must return a valid Result containing the result of the operation, including any informational, warning or error messages generated during the operation.
