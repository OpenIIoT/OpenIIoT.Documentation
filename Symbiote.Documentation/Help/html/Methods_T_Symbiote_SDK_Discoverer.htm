DiscoverT(Object)
Recursively traverses the properties of the specified object and any subordinate objects and returns a list of object instances whose Type is assignable from the specified Type.
DiscoverT(Object, IListT)
Recursively traverses the properties of the specified object instance and searches for properties matching the specified Type, adding them to the specified instance list and returning it.
GetDiscoverableProperties
Returns a IListT containing PropertyInfo objects corresponding to the public, private, instance and static properties of the specified Type which are marked with the DiscoverableAttribute attribute.
