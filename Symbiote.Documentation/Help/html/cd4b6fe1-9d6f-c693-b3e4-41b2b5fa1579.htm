The IConfigurationManager type exposes the following members.

Methods
  NameDescription
AddInstanceConfigurationT
Adds the specified configuration to the specified instance of the specified type.
GetInstanceConfigurationT
Retrieves the configuration for the instance matching instanceName of the supplied Type. If the configuration can't be found, returns the default configuration for the Type.
IsConfigurable
Evaluates the provided type regarding whether it can be configured and returns the result. To be configurable, the type must implement IConfigurable and must have static methods GetConfigurationDefinition and GetDefaultConfiguration.
IsConfigured
Determines whether the specified instance of the specified type is configured.
IsInState
Returns true if any of the specified State s match the current State.
(Inherited from IStateful.)
IsRegistered
Checks to see if the supplied Type is in the RegisteredTypes dictionary.
RegisterType
Registers the supplied Type with the Configuration Manager.
RegisterTypes
Registers each Type within the supplied list which implements the IConfigurable interface.
RemoveInstanceConfiguration
Removes the specified instance of the specified type from the configuration.
Restart
Restarts the stateful object.
(Inherited from IStateful.)
SaveConfiguration
Saves the current configuration to the file specified in app.exe.config.
Start
Starts the stateful object.
(Inherited from IStateful.)
Stop
Stops the stateful object.
(Inherited from IStateful.)
UpdateInstanceConfiguration
Saves the specified Configuration model to the Configuration for the specified instance and Type.
ValidateConfiguration
Validates the current configuration.
