InstallPlugin(PluginArchive, ListIPlugin, IPlatform, Boolean)

Installs the Plugin contained within the supplied PluginArchive using the supplied IPlatform and adds the installed Plugin to the supplied PluginManagerConfiguration.

Prior to installing, the Plugin Archive is re-parsed to ensure it did not changed between the time it was loaded into the PluginArchives list and when installation was requested. If the Plugin within the archive is the same as the loaded plugin, installation continues, otherwise the operation fails and requests that the user refreshes the list.
