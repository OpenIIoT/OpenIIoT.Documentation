Remarks

Only properties and classes marked with the DiscoverableAttribute attribute are traversed.

In order to be discovered, a property must be of a Type assignable from the specified Type, it must be marked with the DiscoverableAttribute attribute, the containing class must be marked with the DiscoverableAttribute attribute, each class within the class hierarchy between the specified object and the discoverable object must be marked with the DiscoverableAttribute, and each property definition used to establish the class hierarchy must be marked with the DiscoverableAttribute.

In layman's terms, the DiscoverT(Object) method uses the DiscoverableAttribute attribute as a "trail of breadcrumbs" to each discoverable object.
