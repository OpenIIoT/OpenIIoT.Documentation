Remarks

Upon the addition of the initial subscriber, an entry is added to the Subscriptions Dictionary keyed with the specified Item with a new ListT of type ActionT containing one entry corresponding to the specified callback delegate.

Successive additions add each of the specified callback delegates to the Subscriptions dictionary.
